NetworkBettina for death and growth dynamics.

This module inherits the tree growth functionality of SimpleBettina, but can account for resource loss and water transfer through root grafting.

def pyMANGA.TreeModelLib.GrowthAndDeathDynamics.NetworkBettina.NetworkBettina.NetworkBettina.rootGraftFormationV0 (   self)

This function handles immediate root graft formation.

That is, if the roots of two trees are in contact (determined by the below-ground concept), they are considered as grafted together without using any resources for the process of graft formation.

def pyMANGA.TreeModelLib.GrowthAndDeathDynamics.NetworkBettina.NetworkBettina.NetworkBettina.rootGraftFormationV1 (   self)

This function reduces growth during the process of root graft formation.

It is assumed that the process will take 2 years (this is subject to change).

def pyMANGA.TreeModelLib.GrowthAndDeathDynamics.NetworkBettina.NetworkBettina.NetworkBettina.rootGraftFormationV2 (   self)

def pyMANGA.TreeModelLib.GrowthAndDeathDynamics.NetworkBettina.NetworkBettina.NetworkBettina.treeGrowthWeightsV2 (   self)

This functions calculates the growths weights for distributing biomass increment to the geometric (allometric) tree measures as defined in SimpleBettina.

In addition, the function calls the root graft formation function, if the tree is currently in the process of root graft formation.
